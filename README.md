# list-maker

###Co potřebuji k nastarování lokální aplikace?

- nodejs
- npm
- mysql (nastartovat local, například přes xampp)

###Nastavení databáze

- databaze: ListMaker
- tabulka: ListInfo
- sloupce v tabulce:
  - id (INT a AI)
  - nameOfItem (VARCHAR)
  - bought (BOOLEAN)
  - listId (VARCHAR)

###Spuštění aplikace

- npm install
  ....po úspěšně instalaci v rootu projektu
- npm start

Aplikace běží na portu 3000
